# 金融多模态融合应用说明

## 🏦 研究背景

### 金融领域的多模态数据特点

金融决策需要整合多种异构信息源：

1. **图像模态**
   - K线图、技术分析图表
   - 财务报表图表
   - 市场趋势可视化
   - 金融数据仪表盘

2. **文本模态**
   - 金融新闻、公告
   - 分析师报告
   - 社交媒体情绪
   - 政策文件解读

3. **数值模态**
   - 价格序列
   - 技术指标（RSI、MACD等）
   - 财务指标（PE、ROE等）
   - 宏观经济数据

### 金融多模态融合的挑战

1. **数据异构性**：不同模态的数据格式、分布差异大
2. **时间敏感性**：金融数据具有强时间依赖性
3. **非线性关系**：金融变量间存在复杂的非线性关联
4. **不确定性**：市场波动、信息不完全
5. **实时性要求**：需要快速处理和分析

## 📊 FinMME数据集

### 数据集描述

**FinMME (Financial Multimodal Evaluation)** 是一个金融多模态问答数据集：

- **任务类型**：多模态问答（4选1）
- **数据模态**：
  - **图像**：金融图表（K线图、技术分析图等）
  - **文本**：问题描述和选项文本
- **数据规模**：训练集和测试集
- **应用场景**：金融知识问答、投资决策支持

### 数据格式

```python
{
    "image_path": "train_000000.jpg",  # 金融图表
    "question": "根据图表，这只股票的趋势是？",  # 问题文本
    "options": ["上涨", "下跌", "震荡", "不确定"],  # 选项
    "answer": "上涨",  # 正确答案
    "label": 0  # 标签（0-3）
}
```

### 使用FinMME数据集

```bash
# 1. 下载数据集
python utils/download_finmme.py --output_dir data/finmme

# 2. 使用FinMME配置训练
python train.py --config configs/config_finmme.yaml
```
